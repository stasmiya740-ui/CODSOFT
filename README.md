## 1️⃣ Personal Portfolio Website

A modern and responsive personal portfolio website built using **HTML, CSS, and JavaScript** to showcase projects, skills, and contact information.

### 🚀 Features

* Responsive Design
* Hero Section
* About Me Section
* Skills Showcase
* Projects Gallery
* Contact Form
* Smooth Navigation

### 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

### 📂 Project Structure

```text
portfolio-website/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

### ⚙️ How to Run

1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in your browser

### 📌 Future Improvements

* Dark Mode
* Resume Download
* Blog Section
* Advanced Animations
